# Nano shortcuts

## Delete
Del line - Alt + Del  
Del word - Ctrl + Del  
Del prev word - Alt + Bsp  

## Copy - pate
Copy line - Alt + 6  
Cut line - Ctrl + K  
Paste - Ctrl + U  

## Do - redo
Undo - Alt + U  
Redo - Alt + E  

## View
Alt + P - Show spaces  
Alt + S - Softwrap  

## Search
Alt + G - Go to line  
Crl + W - Search  
Alt + W - Search next

## Altres
Alt + 3 - Comment/Uncomment  
Ctrl + T - Run command  
Alt + N - Show line numbers  
Alt + A - Seleccionar
